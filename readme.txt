	Awesome & Quick Resume builder.

== Description ==

Create awesome resume.

Easy way to create your CV/Resume/Biodata, predefined fields lets you input unlimited data and display frontend with awesome themes.


### Resume's Builder by http://paratheme.com

* [See the Live demo!&raquo;](http://paratheme.com/demo/resumes-builder/)



<strong>Plugin Features</strong>

* Unlimited input fields.
* Resume thumbnail.
* Predefined fields for Education, Experiences, Contact Info, Award, Skill, Social, Interest, Language, Portfolio.
* Graphical(percentage bar) represantation for skill values.

https://www.youtube.com/watch?v=YXwUFSU23iU

== Installation ==

1. Install as regular WordPress plugin.<br />
2. Go your plugin setting via WordPress Dashboard and find "<strong>Resume's Builder</strong>" activate it.<br />

After activate plugin you will see "Resumes" menu at left side on WordPress dashboard click "New Resumes" and use the options field "Resumes Options"<br />

<br />
<strong>How to use on page or post</strong><br />
When Resumes options setup done please publish Post Grid as like post or page<br />

and then copy shortcode from top of <strong>Resumes Options</strong> `[resumes  id="123456" ]`<br />

then paste this shortcode anywhere in your page to display grid<br />
